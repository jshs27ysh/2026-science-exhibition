Bayesian Optimization for spin coating uniformity.

## Overview

This repository contains Python code for optimizing spin coating conditions using Bayesian Optimization.

The project focuses on improving thin-film uniformity by analyzing image-based uniformity metrics and using them as an objective function for process optimization.

## Research Purpose

Spin coating is widely used in thin-film fabrication, but film uniformity is affected by multiple process variables such as spin speed, coating time, and solution concentration.

This project aims to efficiently search for optimized spin coating conditions with fewer experiments by applying Bayesian Optimization.

## Main Features

- Image-based uniformity analysis
- Calculation of uniformity-related metrics
- Bayesian Optimization for process condition search
- Visualization of optimization results

## Variables

The main process variables considered in this project are:

- Spin speed
- Coating time
- Solution concentration
